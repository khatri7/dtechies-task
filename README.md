# DTechies Task

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in the development mode on port `3000`

### `npm run api`

Runs mock REST API using [json-server](https://github.com/typicode/json-server) using `db.json` located in `src/utils/data` on port `3001`

#### Routes available:

- `/users`
- `/scores`
- `/me`

### `npm run dev`

Runs both react client application and mock REST API simultaneously using [concurrently](https://github.com/open-cli-tools/concurrently)
